# 💈 Barber Shop Management System

A modern web-based salon management system with authentication, customer records, employee tracking, and earnings calculation.

---

## 🚀 Features

- 🔐 Secure Signup with Secret Code
- 🔑 Login System
- 🏪 Multi-Shop Support
- 👥 Employee Management
- 💇 Multiple Services Selection
- 💰 Total Earnings + Employee Earnings
- 📋 Professional Record Table
- 🗑 Delete Records
- 📱 Fully Responsive UI
- ⏳ 30-Day Subscription System

---

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Storage: In-memory (for demo)
